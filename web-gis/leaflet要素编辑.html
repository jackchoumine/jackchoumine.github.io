leaflet 要素编辑 #

使用 leaflet 插件geoman-io实现要素编辑功能，这个插件有免费版和付费版，免费版功能已经很强大了，付费版功能更强大，但是价格也不菲，最低 999 欧元/年。

安装 #

bash
npm i @geoman-io/leaflet-geoman-free
1

引入插件

js
import '@geoman-io/leaflet-geoman-free'
import '@geoman-io/leaflet-geoman-free/dist/leaflet-geoman.css'
1
2

没找到按需引入的方法。

添加编辑控件到地图 #

js
map.pm.addControls({
  position: 'topright',
})
1
2
3

位置可以是：topleft, topright, bottomleft or bottomright

这样就可在地图上绘制要素了。
