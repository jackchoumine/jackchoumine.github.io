  1. 生成随机字符串

场景：希望改变 vue 组件的 key，让元素重新渲染。

js
const randomStr = Math.random().toString(36); // toString 的参数是一个数字，表示进制 2-36
1

为何使用 36，因为 10 个数字何 26 个小写字母的和刚好为 36。

不想含有数字，如何实现？

js
const randomStr = Math.random().toString(36).slice(2);
1

想要包含大写字母，如何实现？ 不能实现。
