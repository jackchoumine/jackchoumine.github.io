# 使用 vitest 测试 vue 组件

## 应该测试什么

* 组件接口：props、事件、slots

* 组件输出：渲染结果、样式、事件

* 副作用：异步请求、定时器、事件监听

> 不要测试组件的内部实现，而是测试组件的行为，测试方法越是接近用户的使用方式越好。
