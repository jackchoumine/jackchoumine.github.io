## 在 ios 模拟器中查看效果

安装依赖

```bash
npm install @capacitor/ios
```

添加 ios 平台

```bash
npx cap add ios
```

运行：

方式一：

```bash
npx cap run ios # 会提示你选择模拟器
```

方式二：在 xcode 中打开运行：

```bash
npx cap open ios # 在 xcode 中打开项目
open ios/App/App.xcworkspace # 或者
```

打开选择模拟器运行
