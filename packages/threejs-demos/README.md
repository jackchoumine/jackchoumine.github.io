# threejs 学习笔记

## 介绍

threejs 是一个在浏览器里绘制3D场景的 js 库。基于 webGL, 即 web 图形库，它封装了诸如场景、灯光、阴影、材质、贴图、空间运算等一系列功能，让你不必要再从底层 WebGL 开始写起。

### threejs 三个要素

> 场景

放置物体的容器，类似舞台。

> 相机

类似人眼，可调整角度、位置，展示场景里的不同位置的不同物体。

> 渲染器

接收场景和相机，在浏览器上渲染出 2D 画面。

为何是 2D 画面？浏览器里支持渲染 2D 图形， threejs 会调整角度，让人看上去是 3D。
